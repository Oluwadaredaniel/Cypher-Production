# CYPHER

Control your entire PC from your phone. 100% local, no cloud.

## What is CYPHER?

CYPHER lets you:
- **Transfer files** between phone and PC (both directions)
- **Control your PC** (power, media, keyboard, mouse)
- **Monitor stats** (CPU, RAM, disk, battery)
- **Sync clipboard** between devices
- **All over WiFi** (no internet required)

## Features

✓ File transfer (download/upload)
✓ Power control (shutdown, restart, sleep, lock)
✓ Media controls (play, pause, volume, skip)
✓ System monitoring (CPU, RAM, disk, battery)
✓ Remote keyboard & mouse
✓ Clipboard sync
✓ Activity logging
✓ 100% local (no servers, no cloud)

## Quick Start

### Prerequisites
- Windows 10+ (PC)
- Android 11+ (Mobile)
- Python 3.12+
- Flutter 3.13+

### Installation

1. Prepare your project directory:
```bash
cd Cypher-Production
Start PC backend:

Bash

python -m core.server
Run mobile app:

Bash
